# About-Myself 蘇盈安的履歷RPG

### 開發時間 : 2019/02/18
### 開發環境 : Unity、Visual Studio 2017
### 使用語言 : C#
### 雲端資料庫 : Arcalet雲端遊戲平台

### 遊戲方法
> 1.使用帳號密碼登入遊戲，可以自己註冊帳號密碼，也可以使用底下的測試帳號密碼登入遊戲。
> 2.利用滑鼠點擊使角色移動到目標位置並完成任務(利用滑鼠右鍵點擊地板，滑鼠左鍵點擊物品)。
> 3.完成任務後可解鎖我-蘇盈安的資料。
